## ✨ Features

- 🖼️ **18+ 이미지 포맷 지원** - JPG, PNG, GIF, WEBP, AVIF, HEIC, SVG, TIFF 등
- 📚 **압축 파일 지원** - ZIP, CBZ, 7Z, RAR, CBR, TAR
- 🎨 **이미지 보정** - 밝기, 대비, 채도, 선명도
- 🤖 **AI 번역** - Gemini, OpenAI, Claude API 연동
- 🌍 **11개 언어** - 한국어, 영어, 일본어, 중국어 등
- 🎭 **4가지 테마** - 라이트, 다크, 두뷰, AMOLED
- 📊 **EXIF/IPTC 메타데이터** 보기
- ⌨️ **키보드 단축키** 지원
- 📌 **북마크 & 읽기 위치 저장**

## ⌨️ Shortcuts

| 단축키 | 기능 | 단축키 | 기능 |
|--------|------|--------|------|
| `Tab` | 사이드바 토글 | `T` | 이미지 번역 |
| `Space` | 스크롤 (끝에서 페이지 전환) | `I` | 이미지 정보 |
| `Shift+Space` | 위로 스크롤 | `F` | 필터 패널 |
| `↑` `↓` | 이미지 이동 | `V` | 보기 모드 전환 |
| `1-9` | 페이지 비율 이동 | `A` | 자동 스크롤 |
| `+` `-` | 확대/축소 | `R` | 이미지 회전 |
| `0` | 확대 초기화 | `B` | 북마크 |
| `H` | 단축키 도움말 | `F11` | 전체화면 |
| `ESC` | 닫기 | | |
